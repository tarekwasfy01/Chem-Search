
# ChemSearch

ChemSearch is a modern desktop application for chemistry researchers, students, laboratories, and educators. It combines molecular structure drawing, chemical database searching, literature discovery, reaction analysis, retrosynthesis, safety information, bioactivity data, and local chemical databases in a single Windows application.

Draw molecules and reactions using the integrated chemical editor or search by chemical name, CAS number, SMILES, InChI, InChIKey, DOI, or reaction. ChemSearch simultaneously searches multiple scientific resources and presents the results in one unified interface.

The application supports public chemistry databases including PubChem, ChEMBL, ChEBI, Rhea, UniProt, Europe PMC, PubMed, Crossref, and OpenAlex. Optional connectors allow integration with licensed services such as CAS SciFinder and Elsevier Reaxys for users with valid institutional access.

## Key Features
Integrated molecular and reaction editor
Search by name, CAS, SMILES, InChI, InChIKey, DOI, or reaction
Structure, substructure, similarity, and identifier searches
Simultaneous search across multiple chemistry databases
Literature and publication search
Bioactivity and protein information
Chemical reaction analysis
Retrosynthesis via configurable local services
Chemical calculators
Safety lookups including GESTIS and other public resources
Local SDF, RDF, JSON, MOL, RXN, JCAMP-DX, and text database support
Research notebook with project management
Import and export of research projects
Secure local storage of connector settings
Optional integration with licensed scientific databases
Modern desktop interface optimized for Windows

ChemSearch is designed for researchers who need a powerful chemistry workstation while keeping full control over their research data. Project files, imported databases, and notes remain on your computer. Searches are performed directly against the selected databases without routing through developer-operated servers.

Whether you are performing compound identification, reviewing literature, investigating biochemical pathways, preparing laboratory work, or organizing research projects, ChemSearch provides a unified environment for modern chemical research.
